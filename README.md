
# Генератор цитат

Простое и красивое React-приложение для генерации случайных цитат, без использования сторонних библиотек.

![Скриншот проекта](public/prob.png)

##  Особенности

*   **Без лишних зависимостей:** Написано на чистом React + CSS.
*   **Адаптивность:** Отлично смотрится на ПК и мобильных устройствах.

## ️ Технологии

*   [React.js](https://reactjs.org/) 
*   CSS3
##  Как запустить

Для запуска проекта на локальной машине выполните следующие шаги:

1.  **Клонируйте репозиторий**
    ```
    git clone https://github.com/your-username/quote-generator.git
    ```

2.  **Перейдите в папку проекта**
    ```
    cd Random_citation_generator
    ```

3.  **Установите зависимости**
    ```
    npm install
    ```

4.  **Запустите приложение**
    ```
    npm run preview
    ```

5. **Откройте ссылку в браузере.**
